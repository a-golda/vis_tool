# vis_tool
After relocating, I started hunting for a place to live. To make the search less chaotic, I wrote a Python tool that automatically pulls rental data from RightMove and Zoopla. The best part - it maps everything onto an area’s deprivation indexes so you can instantly see both price and context.  Super simple, super useful. Feel free to use it!
